----------
```php
＜meta http-equiv="refresh" content="20">?//其中20指每隔20秒刷新一次页面,放到页面的<head>下就好
```


----------
js 版本
```javascript
＜script language="JavaScript">
function myrefresh(){
window.location.reload();
}
setTimeout('myrefresh()',1000); //指定1秒刷新一次
＜/script>
```
